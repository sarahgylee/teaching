# Excel for Data Analytics Instructional Module

### Module Overview
I developed this instructional curriculum to provide a comprehensive introduction to Microsoft Excel as a foundational tool for data analytics. The material focuses on the transition from basic spreadsheet operations to advanced data modeling and business intelligence workflows within the University of Toronto analytics program.

### Core Technical Competencies

* **Data Structuring and Integrity**
Guidance on data types and cleaning techniques to ensure accuracy for extensive datasets.
* **Analytical Modeling with Formulas**
Practical instruction in using logical and statistical functions to derive complex business metrics.
* **Pivot Table Architecture**
Advanced modules on building dynamic summaries and multidimensional data views for rapid insight generation.
* **Visual Narrative Development**
Technical labs on creating informative bar and pie charts to communicate statistical findings to stakeholders.
