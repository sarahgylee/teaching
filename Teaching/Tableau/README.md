# Tableau for Data Visualization Instructional Module

### Module Overview
I developed this instructional curriculum to provide a comprehensive introduction to Tableau as a primary tool for visual analytics and business intelligence. The material focuses on transforming complex health datasets into intuitive visual narratives within the University of Toronto analytics program.

### Core Technical Competencies

* **Visual Analytics Architecture**
Guidance on connecting to diverse data sources and establishing proper data relationships for seamless visualization.
* **Geospatial Mapping and Analysis**
Practical instruction on building interactive maps to visualize regional data distribution and identifying geographical trends.
* **Dynamic Dashboard Design**
Advanced modules on creating integrated dashboards that allow stakeholders to filter and drill down into specific data segments.
* **Narrative Data Storytelling**
Technical labs focused on utilizing visual best practices to communicate statistical findings to non technical audiences.

### Case Study COVID 19 Vaccine Analytics
The module features a real world dataset regarding vaccine administration in Canada where students apply visualization techniques to report on public health progress.
